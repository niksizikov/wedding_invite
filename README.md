# Wedding Invite

Статический лендинг-приглашение Елизаветы и Никиты, готовый для публикации через GitHub Pages.

## Как опубликовать на GitHub Pages

1. Загрузите файлы в репозиторий GitHub.
2. Откройте `Settings` -> `Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
5. Сохраните настройки и дождитесь публикации.

## Что заменить перед публикацией

- Финальные детали площадки: ресторан «Мишель», Москва, ул. Красная Пресня, 13.
- Ссылку на карту при необходимости можно заменить на точную ссылку из Яндекс Карт.
- Рабочий тайминг программы.
- Ссылку `https://forms.gle/REPLACE_WITH_YOUR_FORM` на реальную Google Форму в блоке «Анкета гостя».
- Дресс-код и детали для гостей.
